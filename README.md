# 🌐 Personal Portfolio Website

Welcome to my personal portfolio repository! This is a modern, highly interactive web experience built to showcase my projects, design work, and development skills.

## 🛠️ The Tech Stack

This project was built using a modern React ecosystem, focusing on high-performance animations and utility-first styling:

* **Framework:** Next.js (React)
* **Styling:** Tailwind CSS
* **Animation Engine:** GSAP (GreenSock) & ScrollTrigger
* **Deployment:** Vercel

## ✨ Key Features

* **Advanced Scroll Animations:** Utilizes GSAP's `ScrollTrigger` for complex, timeline-based scroll sequences.
* **3D Hardware Showcase:** Features a custom MacBook scaling and rotation animation that reveals dynamic video content based on active project tabs.
* **Fully Responsive:** Carefully calculated viewport units and Tailwind breakpoints ensure the layout and GSAP math work seamlessly across mobile, tablet, and desktop.
* **Modern UI/UX:** Clean typography, gradient overlays, and smooth reveal staggered components.

## 🚀 Getting Started Locally

Want to run this project on your own machine? Follow these steps:

### Prerequisites

* Node.js (v18 or higher recommended)
* npm, yarn, or pnpm

### Installation

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/markappscript-markss/portfolio.git](https://github.com/markappscript-markss/portfolio.git)
   ```

2. **Navigate to the directory:**
   ```bash
   cd portfolio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the local development server:**
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser to see the animations in action!
